# Requirement: 쿠폰 자동 배포 & 추천인 활성화

## 개요
신규 사용자 획득을 위한 추천인(referral) 시스템과, 2주마다 자동으로 쿠폰을 지급하는 배포 로직.

## 배경
음성 녹음(260319_155013.m4a)에서 "설치 후 1주일 안에 지인 2명을 추천하면 3,000원 쿠폰을 받는다. 이후 2주마다 자동 지급"이라고 명시.  
현재 앱에는 `Coupon.swift` 모델과 `User.swift`의 `referralCode` 필드만 존재하며, 비즈니스 로직이 없다.

## 범위

### In Scope
- 추천인 코드 공유 UI
- 추천 완료 조건 판별: 피추천인이 앱 설치 후 1주일 내 2명 추천 달성 시
- 조건 달성 시 3,000원 쿠폰 자동 지급
- 이후 2주 주기 쿠폰 자동 배포
- 쿠폰 목록 / 만료일 표시 UI

### Out of Scope
- 쿠폰으로 실제 결제 연동 (별도 결제 requirement 참조)
- Push notification 발송 로직 (별도 NotificationManager 활용)

## 기능 요구사항

### FR-1. 추천인 코드 공유
- 사용자는 마이페이지에서 자신의 추천인 코드를 확인하고 공유할 수 있다.
- 공유 방법: 클립보드 복사, iOS 공유 시트

### FR-2. 추천인 등록
- 신규 사용자는 앱 가입 시 (온보딩 흐름 중) 추천인 코드를 입력할 수 있다.
- 입력된 코드는 Supabase에 연결되어 추천인의 `referral_count`를 증가시킨다.

### FR-3. 1주일 내 2명 조건 달성 감지
- 가입일로부터 7일 이내에 `referral_count >= 2` 달성 시 쿠폰 1개(3,000원) 자동 지급.
- 7일 초과 시 조건 미달로 처리 (쿠폰 미지급, 추천 카운트 리셋).

### FR-4. 2주 주기 자동 쿠폰 배포
- 조건 달성 후 매 14일마다 3,000원 쿠폰 1개 자동 지급.
- 배포 주체: Supabase Scheduled Function (cron) 또는 Edge Function 트리거.

### FR-5. 쿠폰 목록 UI
- 마이페이지에서 보유 쿠폰 목록 표시 (금액, 만료일, 사용 여부).
- 만료된 쿠폰은 별도 섹션으로 구분.

## 비기능 요구사항
- 쿠폰 지급 로직은 클라이언트가 아닌 서버(Supabase Edge Function)에서 처리해 위변조 방지.
- 추천인 코드는 중복 불가, 대소문자 무관 처리.
