# Requirement: 모의투자 (Mock Investment)

## 개요
투자 성향 분석 결과를 기반으로 사용자가 실제 돈 없이 주식/ETF를 가상으로 투자해보는 시뮬레이션 기능.

## 배경
음성 녹음(260319_155013.m4a)에서 의뢰인이 "사주 분석으로 나온 투자 성향에 맞게 모의투자를 할 수 있어야 한다"고 명시.  
현재 앱에는 InvestmentProfile 모델과 교육 콘텐츠만 있으며, 실제 시뮬레이션 기능은 없다.

## 범위

### In Scope
- 투자 성향 분석 결과(공격형/안정형/가치형/성장형)에 맞는 종목 추천 (ETF 중심)
- 가상 포트폴리오 구성: 종목 선택 + 비중 설정
- 과거 수익률 시뮬레이션 (예: "이 포트폴리오를 10년 전에 구성했다면?")
- 포트폴리오 저장 및 조회
- 구독 티어별 접근 제한 (Basic 이상)

### Out of Scope
- 실제 증권사 연동 / 실매매
- 실시간 시세 조회
- 단일 종목 개별 분석

## 기능 요구사항

### FR-1. 성향 기반 포트폴리오 추천
- 시스템은 사용자의 `InvestmentType`에 따라 추천 포트폴리오(종목 + 비중)를 1개 이상 제시한다.
- 추천 종목은 ETF 우선 (QQQ, VOO, KODEX 등)

### FR-2. 가상 투자 시작
- 사용자는 가상 시드 금액을 설정한다 (예: 100만원, 500만원, 1,000만원).
- 추천 포트폴리오를 그대로 사용하거나, 비중을 직접 조정할 수 있다.

### FR-3. 수익률 시뮬레이션
- 설정된 포트폴리오에 대해 과거 1년 / 3년 / 5년 / 10년 수익률을 계산해 표시한다.
- "현재 시드를 10년 전에 투자했다면 얼마?" 형태의 결과 카드를 제공한다.

### FR-4. 포트폴리오 저장
- 사용자는 최대 3개까지 포트폴리오를 저장할 수 있다 (Free 티어).
- Basic 이상은 무제한 저장.

### FR-5. 구독 게이트
- Free 티어: 추천 포트폴리오 조회까지 허용, 시뮬레이션 실행 불가 (업그레이드 유도 CTA 표시)
- Basic 이상: 모든 기능 사용 가능

## 비기능 요구사항
- 수익률 데이터는 앱 내 정적 데이터 또는 Supabase Edge Function으로 제공 (실시간 API 불필요)
- 계산 결과는 투자 조언이 아님을 명시하는 면책 문구 필수
