# 01. 온보딩 플로우

## 목적
사용자가 앱을 처음 설치한 후 사주 분석에 필요한 기본 정보를 입력하게 한다.

## 화면 흐름

### 1. 스플래시 화면
- 앱 로고 + 간단한 로딩

### 2. 회원가입/로그인
- 소셜 로그인 (카카오, 애플)
- 전화번호 인증 (선택)

### 3. 생년월일 입력
- 생년월일 (날짜 피커)
- 양력/음력 선택
- 태어난 시간 (12시진 선택, "모름" 허용)
- 성별 선택

### 4. 추천인 코드 입력 (선택)
- 추천 코드 입력 필드
- "건너뛰기" 가능

### 5. 첫 운세 결과
- 입력 즉시 오늘의 사주 요약 표시
- "더 자세히 보기" → 메인 화면으로

## 타겟 사용자
- 20~50대 여성
- 사주/운세에 관심이 있는 사용자

## Supabase 연동
- `supabase.auth.signUp()` → users 테이블 자동 생성 (trigger)
- 생년월일/시/성별 → `users` 테이블 UPDATE
- 추천 코드 → `referrals` 테이블 INSERT
