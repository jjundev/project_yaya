# 02. 운세 기능

## 목적
AI를 활용하여 사용자의 사주 기반 맞춤형 운세를 제공한다.

## 기능 범위

### 무료: 오늘의 운세
- 매일 갱신되는 일일 운세
- 사랑/재물/건강/직장 점수 (1~5)
- 행운의 숫자, 행운의 색
- 오늘의 조언

### Basic (4,900원/월): + 주간 운세
- 이번 주 전체 흐름 분석
- 요일별 주의사항

### Standard (9,900원/월): + 월간 운세
- 이번 달 전체 운세 흐름
- 월간 재물운 상세 분석

### Premium (19,000원/월): + 연간 운세 + AI 상담
- 올해 전체 운세
- 1:1 AI 사주 상담 (채팅)

## 사주 분석 (공통 기반)

### 입력
- 생년월일 (양력/음력)
- 태어난 시 (12시진, "모름" 허용)
- 성별

### AI 분석 항목
- 사주 요약
- 성격 분석 (장점/단점)
- 재물운
- 대인관계
- 적합한 직업/진로
- 오행 비율 (목/화/토/금/수)

## 기술 구현

### Supabase Edge Function
- `calculate-saju`: 사주팔자 분석 (최초 1회, 결과 캐시)
- `daily-fortune`: 운세 생성 (매일 호출, 날짜별 캐시)

### 캐싱 전략
- `fortunes` 테이블에 날짜+타입별 캐시
- 동일 날짜 재접속 시 캐시된 운세 반환
- AI API 비용 절감 효과

### 푸시 알림
- 매일 아침 8시 오늘의 운세 요약 푸시 발송
