# 03. 투자 성향 분석

## 목적
사주 분석 결과를 기반으로 사용자의 투자 성향을 파악하고, 맞춤형 투자 교육으로 연결한다.

## 4가지 투자 성향

### 1. 공격형 (Aggressive) 🔥
- **사주 특성**: 화(火)·목(木)이 강한 사주, 비견·겁재 많음
- **투자 스타일**: 리스크 감수, 높은 수익 추구
- **추천 ETF**: QQQ, TQQQ, ARKK
- **교육 방향**: 리스크 관리, 분산투자의 중요성

### 2. 안정형 (Stable) 🛡️
- **사주 특성**: 토(土)·금(金)이 강한 사주, 정관·정인 많음
- **투자 스타일**: 안전한 자산, 꾸준한 수익
- **추천 ETF**: VOO, VTI, BND
- **교육 방향**: 복리 효과, 장기투자 전략

### 3. 가치투자형 (Value) 💎
- **사주 특성**: 수(水)·금(金)이 강한 사주, 편인·식신 많음
- **투자 스타일**: 기업 본질 분석, 저평가 종목
- **추천 ETF**: VTV, SCHD, BRK.B
- **교육 방향**: 재무제표 읽기, 가치평가 방법

### 4. 스타트업 추구형 (Growth) 🚀
- **사주 특성**: 목(木)·화(火)가 강한 사주, 상관·식신 많음
- **투자 스타일**: 혁신 기업, 성장 가능성 투자
- **추천 ETF**: VUG, SOXX, IGV
- **교육 방향**: 산업 트렌드 분석, 성장주 선별

## 복리 계산기
- 투자 원금, 기간, 예상 수익률 입력
- 세금 22% 적용 후 실수령액 계산
- VOO/QQQ 과거 수익률 기반 시뮬레이션

## 기술 구현

### Supabase Edge Function
- `investment-personality`: 사주 → 투자 성향 매칭

### 데이터 모델
- `investment_profiles` 테이블에 분석 결과 저장
- 사용자당 1개 프로필 (재분석 시 업데이트)
