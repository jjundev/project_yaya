# 오늘의 운세 상세 화면 스펙

## 개요

`FortuneHomeView`의 오늘의 운세 카드는 핵심 요약(점수, 행운 아이템, 짧은 조언)만
노출한다. 사용자가 더 깊은 인사이트를 원할 때 들어갈 수 있는 전용 상세 화면이 없어,
앱의 핵심 가치인 "AI 사주 기반 개인화 운세"가 충분히 전달되지 못하고 있다.
이 화면은 영역별(사랑·재물·건강·직장) 심층 해설, AI가 작성한 오늘의 개인화 편지,
행운 아이템 모음을 한 곳에서 보여줌으로써 매일 재방문 동기를 강화한다.

## 범위

### 포함
- `FortuneHomeView` 오늘의 운세 카드 → 탭 → 상세 화면 진입
- 오늘 날짜 및 전체 운세 요약 헤더
- 영역별 상세 해설 섹션: 사랑운 / 재물운 / 건강운 / 직장운
  - 각 영역: 점수(1~5) + AI 생성 해설 텍스트
- 오늘의 AI 개인화 메시지 (사주 기반, 편지 형식)
- 행운 아이템 모음: 행운의 숫자, 행운의 색, 행운의 방향, 행운의 시간대
- 공유 기능: 오늘의 운세 카드를 이미지로 저장/공유

### 미포함 (다음 스프린트)
- 주간/월간/연간 운세 상세 화면 (별도 스펙)
- AI 실시간 채팅 상담
- 과거 운세 히스토리 열람
- 행운의 방향·시간대 데이터 모델 추가 (현재 FortuneContent에 없음 → 별도 스프린트)
- 공유 이미지 커스터마이징

## 사용자 스토리

### 진입
- 사용자는 `FortuneHomeView`의 오늘의 운세 카드를 탭하여 상세 화면으로 이동할 수 있다
- 사용자는 상세 화면 상단의 뒤로가기로 홈으로 돌아갈 수 있다

### 헤더
- 사용자는 오늘 날짜와 전체 운세 한 줄 요약을 화면 최상단에서 확인할 수 있다

### 영역별 상세 해설
- 사용자는 사랑운·재물운·건강운·직장운 각 영역의 점수(1~5)를 볼 수 있다
- 사용자는 각 영역에 대한 AI 생성 해설 텍스트(2~4문장)를 읽을 수 있다

### AI 개인화 메시지
- 사용자는 자신의 사주를 기반으로 AI가 작성한 오늘의 편지(개인화 메시지)를 읽을 수 있다
- **[AI 기회]** 메시지는 사용자의 생년월일·출생 시간·오행 구성을 반영해 매일 다르게 생성된다

### 행운 아이템
- 사용자는 오늘의 행운의 숫자와 행운의 색을 확인할 수 있다

### 공유
- 사용자는 오늘의 운세 요약을 이미지로 저장하거나 공유할 수 있다

## 참고 사항

- **기존 데이터 재사용**: `FortuneContent.detailedAnalysis` (현재 미사용 String?)을
  영역별 해설 + AI 편지 데이터 저장에 활용하거나, FortuneContent에 필드 추가
- **구독 제한 없음**: 상세 화면은 Free 사용자도 접근 가능 (오늘의 운세는 무료 콘텐츠)
- **네비게이션**: SwiftUI NavigationLink 또는 sheet 방식 선택은 generator 판단
- **디자인 톤**: 기존 FortuneHomeView의 보라색 그라데이션 + 카드 스타일 일관성 유지
